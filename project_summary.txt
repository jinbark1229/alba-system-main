# 알바생/사장님 통합 관리 시스템 - 상세 기능 명세서

## 1. 프로젝트 개요
이 프로젝트는 매장을 효율적으로 운영하기 위해 **알바생의 근무 관리**와 **사장님의 매장 관리**를 통합한 웹 애플리케이션입니다.
- **기술 스택**: Vite, React, TypeScript, React Router v7
- **디자인 원칙**: 직관적인 블루/화이트 테마, 반응형 모바일 친화적 UI

---

## 2. 사용자별 핵심 기능

### 🧑‍🍳 알바생 (Worker) 기능

**1. 근무 일지 작성 (Daily Log)**
- **기능**: 출근/퇴근 시간과 휴게 여부를 분 단위로 기록합니다.
- **특징**:
  - '휴게 시간 있음' 체크 시 구체적인 휴게 시간(분)을 입력할 수 있습니다.
  - 작성 즉시 하단 목록에 **최신순**으로 업데이트되어 표시됩니다.
  - **실명 기록**: 로그인한 본인의 이름이 시스템에 자동 저장됩니다.

**2. 시간표 조회 (Schedule View)**
- **기능**: 사장님이 올린 이번 주 시간표를 달력 형태로 확인합니다.
- **특징**:
  - **내 시간표 필터링**: 내 이름이 포함된 근무만 쏙 뽑아서 볼 수 있습니다.
  - **다중 매장 탭**: '1호점'과 '2호점' 탭을 클릭하여 내가 일하는 매장의 시간표를 즉시 전환하여 확인합니다.

**3. 근무 변경/대타 요청 (Shift Swap)**
- **기능**: 시간표 페이지 하단 댓글 게시판을 통해 동료들과 소통합니다.
- **특징**:
  - **매장별 분리**: 1호점 게시판과 2호점 게시판이 독립적으로 운영되어 글이 섞이지 않습니다.
  - **삭제 권한**: 내가 쓴 글은 직접 삭제할 수 있습니다.

**4. 기타 편의 기능**
- **급여 계산기**: 시급과 근무 시간을 입력하여 예상 월급을 미리 계산해볼 수 있습니다.
- **공지사항 확인**: 사장님이 올린 공지사항을 읽을 수 있습니다.

---

### 👔 사장님 (Boss) 기능

**1. 시간표 관리 및 업로드**
- **기능**: 엑셀(CSV)로 정리된 시간표를 드래그 앤 드롭으로 한번에 업로드합니다.
- **특징**:
  - **복잡한 양식 지원**: 행(이름)과 열(날짜)로 된 매트릭스 형태의 시간표도 똑똑하게 인식합니다.
  - **소수점 시간 변환**: '8.5'라고 적혀있으면 자동으로 '08:30'으로 변환하여 등록합니다.
  - **매장 선택**: 업로드 시 라디오 버튼으로 '1호점/2호점'을 선택하여 각 매장에 맞게 저장합니다.

**2. 근무 일지 다운로드 (ZIP Export)**
- **기능**: 알바생들이 작성한 근무 일지를 엑셀 파일로 내려받습니다.
- **특징**:
  - **ZIP 압축**: 버튼 한 번으로 모든 데이터를 `근무일지_기간.zip` 파일로 압축해 받습니다.
  - **개별 파일 생성**: `홍길동_근무일지.csv`, `김철수_근무일지.csv` 처럼 사람별로 파일이 따로 생성됩니다.
  - **자동 정렬**: 각 파일 내부 데이터는 '날짜순'으로 깔끔하게 정렬되어 있습니다.

**3. 전지적 관리자 시점**
- **권한 확대**: 원래 알바생만 볼 수 있던 [시간표 페이지]와 [공지사항 페이지]에 사장님도 접근 가능합니다.
- **댓글 관리**: 알바생들이 쓴 댓글을 관리(삭제)할 수 있습니다.
- **공지사항 작성**: 공지사항을 작성하고 삭제할 수 있는 관리자 권한을 가집니다.

## 3. 시스템 및 기술적 특징

### 🔐 보안 및 인증
- **역할 기반 라우팅 (Protected Routes)**:
  - 알바생은 '데이터 내보내기' 페이지에 접근할 수 없습니다.
  - 로그인하지 않은 사용자는 메인 기능에 접근할 수 없습니다.
- **회원가입 분리**: 가입 시점부터 역할을 '알바생'과 '사장님'으로 명확히 구분하여 가입합니다.

### 💾 데이터 보존 (Persistence)
- **LocalStorage 활용**: 서버가 없어도 브라우저 저장소를 활용해 데이터를 영구 보존합니다.
  - 새로고침을 하거나 컴퓨터를 껐다 켜도 작성한 **근무 일지**, **댓글**, **업로드한 시간표**가 사라지지 않습니다.

### 🎨 UI/UX 디자인
- **일관된 디자인 시스템**: 모든 버튼(`btn-primary`, `btn-secondary`)과 입력폼에 통일된 스타일 가이드를 적용했습니다.
- [x] **내비게이션**: 모든 페이지에 '홈으로', '뒤로가기', '로그아웃' 버튼 배치로 이동 편의성 증대.
- [x] **회원탈퇴**: 메인 대시보드에서 안전한 확인 절차를 거쳐 계정을 영구 삭제할 수 있습니다.
- [x] **엑셀 지원**: CSV 한글 깨짐 문제 해결을 위해 XLSX 파일을 직접 업로드할 수 있습니다.
