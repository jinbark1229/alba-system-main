# 🏪 알바시스템 (Alba System)

## 소개

**알바시스템**은 소규모 매장(카페, 음식점, 편의점 등)을 운영하는 사장님과 아르바이트 직원을 위한 **종합 근무 관리 웹 애플리케이션**입니다.

### 이런 분들을 위해 만들었습니다

- 📝 **알바생**: "오늘 몇 시부터 몇 시까지 일했지?" 매일 근무 기록을 남기고 싶은 분
- 💰 **알바생**: "이번 달 월급이 얼마나 되지?" 예상 급여를 미리 확인하고 싶은 분
- 📅 **알바생**: "다음 주 내 스케줄이 뭐지?" 근무 일정을 한눈에 보고 싶은 분
- 👔 **사장님**: "이번 달 누가 얼마나 일했나?" 직원별 근무 기록을 정리하고 싶은 분
- 🔐 **사장님**: "아무나 가입하면 안 되는데..." 승인된 직원만 시스템을 사용하게 하고 싶은 분

## ✨ 핵심 기능 상세 설명

### 1️⃣ 인증코드 기반 회원가입 시스템

> **"사장님이 발급한 인증코드만 가입할 수 있습니다"**

일반적인 웹사이트처럼 아무나 회원가입할 수 없습니다. 사장님이 먼저 직원을 등록하고 고유한 **인증코드**를 발급해야 회원가입이 가능합니다.

**작동 방식:**
1. 사장님이 "직원 관리" 페이지에서 직원 이름 등록 (예: "홍길동", 역할: 알바생)
2. 시스템이 자동으로 고유 **인증코드** 발급 (예: `AbCdEfGhIjKlMn`)
3. 사장님이 홍길동에게 해당 인증코드를 전달
4. 홍길동이 회원가입 페이지에서 전달받은 **인증코드**를 입력
5. 코드가 유효하면, 이름과 역할이 자동으로 불러와지며 가입 진행 (비밀번호 설정)

**퇴사한 직원/권한 회수 처리:**
- 등록 목록에서 직원을 제거하면 해당 직원은 더 이상 로그인 불가
- 기존에 작성한 근무 일지 데이터는 데이터베이스에 안전하게 보존됨

---

### 2️⃣ 근무 일지 작성 (알바생 전용)

> **"오늘 근무한 시간을 기록하세요"**

매일 출근/퇴근 시간을 기록하여 근무 기록을 남길 수 있습니다.

**입력 항목:**
- 📅 **날짜**: 근무한 날짜 선택
- ⏰ **시작 시간**: 출근 시간 (예: 09:00)
- ⏱️ **종료 시간**: 퇴근 시간 (예: 18:00)
- ☕ **휴게 시간**: 휴게 시간 여부 및 시간 (분 단위)

**예시:**
```
2024년 12월 22일
출근: 09:00 | 퇴근: 18:00 | 휴게: 60분
→ 실제 근무시간: 8시간
```

---

### 3️⃣ 급여 계산기 (알바생 전용)

> **"이번 달 예상 급여를 확인하세요"**

기록된 근무 일지를 바탕으로 월별 예상 급여를 자동 계산합니다.

**계산 방식:**
- 시급 × 총 근무시간 = 예상 급여
- 기본 시급: 2024년 최저시급 9,860원 (수정 가능)
- 휴게 시간은 자동으로 제외됨

**표시 정보:**
- 선택한 월의 총 근무 시간
- 예상 급여 금액
- 해당 월 근무 내역 목록

> ⚠️ 주휴수당, 세금 등은 포함되지 않은 순수 근무시간 기준 예상 금액입니다.

---

### 4️⃣ 근무 일정 캘린더 (알바생 & 사장님)

> **"내 근무 일정을 캘린더로 확인하세요"**

월별 캘린더 형태로 근무 일정을 확인할 수 있습니다.

**기능:**
- 📅 월별 캘린더 보기
- 🏪 1호점/2호점 탭으로 매장 선택
- 각 날짜에 근무 시간 표시 (예: "09:00~18:00")

**대타/근무 변경 요청:**
- 근무 변경이 필요한 경우 댓글 형태로 요청 가능
- 예: "12/25 대타 구합니다!"

---

### 5️⃣ 공지사항 (알바생 & 사장님)

> **"중요한 공지를 전달하고 확인하세요"**

**사장님 기능:**
- 공지사항 작성 (제목, 내용)
- 공지사항 삭제

**알바생 기능:**
- 공지사항 목록 확인
- 공지 내용 열람

---

### 6️⃣ 데이터 내보내기 (사장님 전용)

> **"근무 기록을 엑셀 파일로 다운로드하세요"**

특정 기간의 근무 기록을 CSV 파일 형태로 다운로드할 수 있습니다.

**내보내기 형식:**
- ZIP 파일 (직원별 개별 CSV 파일 포함)
- CSV 내용: 이름, 날짜, 출근시간, 퇴근시간, 휴게여부, 휴게시간

**활용 예시:**
- 월말 정산용 근무 기록 정리
- 급여 계산을 위한 데이터 추출
- 근무 기록 백업

---

### 7️⃣ 허용 이름(인증코드) 관리 (사장님 전용)

> **"직원을 등록하고 인증코드를 발급하세요"**

**기능:**
- ➕ 직원 추가: 이름, 역할(알바생/매니저), 소속 매장 지정 → **자동으로 회원가입용 인증코드 생성**
- ➖ 직원 제거: 시스템에서 삭제 (로그인 즉시 차단)
- 🔄 코드 재발급: 직원이 코드를 분실한 경우 새로운 인증코드 재발급
- 👀 상태 확인: 가입 완료/미가입 상태 표시

**역할 설명:**
| 역할 | 설명 |
|------|------|
| 알바생 | 근무 일지 작성, 급여 확인, 일정 확인 가능 |
| 사장님 | 위 모든 기능 + 관리자 기능 사용 가능 |

## 🚀 설치 및 실행 방법

### 사전 요구사항
- Node.js 18 이상
- npm 또는 yarn

### 설치
```bash
# 프로젝트 폴더로 이동
cd 알바시스템

# 의존성 설치
npm install
```

### 개발 서버 실행
```bash
npm run dev
```
→ 브라우저에서 `http://localhost:5173` 접속

### 프로덕션 빌드
```bash
npm run build
```
→ `dist/` 폴더에 빌드 결과물 생성

### 린트 검사
```bash
npm run lint
```

## � 사용 시나리오

### 시나리오 1: 신규 직원 등록 및 가입

1. **사장님**: 로그인 후 "허용 이름 관리" 페이지 접속
2. **사장님**: "김철수" 이름을 "알바생" 역할로 등록 → **인증코드 발급 (예: XaY12bC)**
3. **사장님**: 김철수에게 발급된 인증코드 전달
4. **김철수**: 회원가입 페이지 접속, **인증코드(XaY12bC)** 입력
5. **시스템**: 코드가 유효함을 확인하고 자동으로 이름("김철수") 세팅. 비밀번호 입력 후 가입.
6. **김철수**: 알바생으로 로그인 성공!

### 시나리오 2: 퇴사 직원 처리

1. **사장님**: "허용 이름 관리" 페이지 접속
2. **사장님**: "김철수" 옆 "제거" 버튼 클릭
3. **시스템**: 확인 메시지 표시 ("로그인 불가, 근무 기록 보존")
4. **사장님**: 확인
5. **김철수**: 다음 로그인 시도 → "더 이상 로그인 권한이 없습니다"

### 시나리오 3: 월말 급여 정산

1. **사장님**: "데이터 내보내기" 페이지 접속
2. **사장님**: 시작일 "2024-12-01", 종료일 "2024-12-31" 선택
3. **사장님**: "ZIP 내보내기" 클릭
4. **시스템**: 직원별 근무 기록이 담긴 ZIP 파일 다운로드
5. **사장님**: 엑셀에서 열어 급여 계산
