# 🏪 알바시스템 (Alba System)

**포트폴리오 프로젝트** | 소규모 매장의 알바생 관리를 위한 종합 근무 관리 웹 애플리케이션

� **[라이브 데모 보기](https://alba-system.vercel.app)** | 아래 테스트 계정으로 모든 기능 체험 가능

---

## 📌 프로젝트 소개

연산점, 부전점 두 개의 지점을 운영하는 사장님과 각 지점의 알바생이 사용하는 근무 관리 시스템입니다.

- 사장님은 **지점별 공지사항 발송**, **근무 일정 업로드**, **직원 관리** 가능
- 알바생은 자신의 **지점에 맞는 일정과 공지만 표시**, **근무 일지 작성**, **급여 계산** 가능
- **역할 기반 접근 제어(RBAC)** 로 메뉴와 기능이 계정마다 다르게 표시됨

---

## 🧪 포트폴리오 테스트 계정

| 역할 | 아이디 | 비밀번호 | 설명 |
|------|--------|---------|------|
| � 관리자 | `admin` | `admin` | 전체 시스템 권한 |
| 👔 사장님 | `boss` | `boss` | 공지 작성·지점 관리 |
| 🍵 연산점 알바 | `yeonsan` | `yeonsan` | 연산점 일정·공지만 열람 |
| 🚌 부전점 알바 | `bujeon` | `bujeon` | 부전점 일정·공지만 열람 |
| 🔄 양쪽 알바 | `dual` | `dual` | 연산+부전 두 지점 모두 열람 |

> 로그인 페이지 하단 "포트폴리오 테스트 계정" 에서 한 번의 클릭으로 자동 입력됩니다.

## ✨ 역할별 기능 상세 가이드

### 👑 관리자 (Admin)

시스템 전체를 관리하는 최상위 권한 계정입니다.

| 기능 | 설명 |
|------|------|
| **사용자 관리** | 전체 계정 목록 조회, 개별 계정 삭제 |
| **허용 이름 관리** | 회원가입 가능한 이름 등록·삭제, 인증코드 재발급 |
| **근무 일정 확인** | 연산점·부전점 전체 일정 캘린더/표 보기 |
| **공지사항 확인·작성·삭제** | 모든 지점의 공지를 읽고 새 공지 작성 및 삭제 |
| **데이터 내보내기** | 기간 선택 후 전체 근무 기록 CSV/ZIP 다운로드 |

---

### 👔 사장님 (Boss)

두 지점(연산점·부전점)을 총괄 운영하는 사장님 계정입니다.

| 기능 | 설명 |
|------|------|
| **공지사항 작성** | 대상 지점 선택(연산점 / 부전점 / 양쪽) 후 공지 등록<br>양쪽 대상 선택 시 **자동으로 "중요" 우선순위 지정** |
| **공지사항 삭제** | 작성된 공지사항 삭제 |
| **공지 지점 뱃지 확인** | 공지 목록·상세 화면에서 "연산점 / 부전점 / 전체" 뱃지로 구분 확인 |
| **근무 일정 확인** | 연산점·부전점 두 지점의 일정을 토글로 전환하며 확인 |
| **근무 일정 CSV 업로드** | Excel에서 내보낸 CSV 파일로 지점별 일정 일괄 등록 |
| **허용 이름 관리** | 회원가입 가능한 이름 등록·삭제, 인증코드 발급 및 재발급 |
| **데이터 내보내기** | 기간 선택 후 근무 기록 CSV/ZIP 다운로드 |

---

### 🧑‍💼 알바생 (Worker)

각 지점에 소속된 아르바이트 직원 계정입니다. 소속 지점에 따라 보이는 정보가 다릅니다.

| 기능 | 설명 |
|------|------|
| **근무 일지 작성** | 날짜·출근시간·퇴근시간·휴게시간 입력 후 기록 저장 |
| **근무 일지 삭제** | 잘못 입력한 기록 삭제 |
| **급여 계산** | 기록된 근무 일지 기반으로 월별 예상 급여 자동 계산 |
| **근무 일정 확인** | 소속 지점 일정만 표시 (지점 토글 없음) |
| **공지사항 확인** | 자기 지점 + 전체 공지사항 열람 (작성·삭제 불가) |
| **비밀번호 변경** | 개인 설정에서 비밀번호 변경 |
| **회원 탈퇴** | 계정 삭제 |

**지점별 일정·공지 노출 규칙:**

| 계정 종류 | 일정 | 공지 |
|-----------|------|------|
| 연산점 알바 (`yeonsan`) | 연산점 일정만 | 연산점 + 전체 공지 |
| 부전점 알바 (`bujeon`) | 부전점 일정만 | 부전점 + 전체 공지 |
| 양쪽 알바 (`dual`) | 연산·부전 모두 (토글) | 모든 공지 |
